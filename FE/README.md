# D4U Frontend

Frontend React cho D4U MVP, dùng cho Phase 1 và Phase 2.

## Tech Stack

- React
- Vite
- React Router
- Ant Design
- Axios

## Run Local

Backend cần chạy trước tại:

```text
http://localhost:8080
```

Chạy frontend:

```powershell
cd D:\Codex\FE
npm install
npm run dev
```

Mở:

```text
http://localhost:5173
```

Frontend gọi backend qua proxy:

```text
/api -> http://localhost:8080
```
